# WebCrawler爬蟲
## 高鐵自動訂票
### 項目簡介
> 直接輸入時間地點快速得到車次資料,
> 後續訂票繁瑣手動流程交給selenium,
>

- 1.輸入乘車時間 起迄站
- 2.顯示出查詢時間之後車次列表
- 3.選擇要訂票的車次
### 功能演示
![](./demo/time_table.png)

- 4.選擇車次並輸入個人資料
- 5.後續由selenium自動完成訂票流程
